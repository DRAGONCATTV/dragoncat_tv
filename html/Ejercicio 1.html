ESTOS SON LOS TIPOS DE META TAGS
1. X-UA-compatible en html es otro meta-tag útil en la página HTML y este meta-tag se relaciona mayormente con el navegador. Dependiendo de los navegadores de Microsoft que usted apoye, puede que no necesite seguir usando la etiqueta X-UA-Compatible. En nuestro caso lo vamos a centrar en el último navegador de Microsoft que es EDGE
2. Viewport: es una de las etiquetas más representativas de la web móvil, que nos permite configurar cómo debe interpretar una página el navegador web para móviles. A. width=device-width: conseguimos que el viewport sea igual a la anchura real de la pantalla del dispositivo, de modo que no se tratará de emular una pantalla mayor de lo que realmente es y veremos los píxeles reales. B. initial-scale=1 conseguimos que no se haga zoom sobre el documento. Es bien simple, el contenido de la web no se transformará, ni se agrandará, ni se hará menor.
3. Title: Además de dar nombre a la propia página nos ayuda a mejorar el posicionamiento SEO.
